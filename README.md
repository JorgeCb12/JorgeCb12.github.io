# 🚀 Modern Frontend Portfolio - Jorge Corrales

Welcome to my professional portfolio. This project is a showcase of my skills as a **Frontend Developer**, focusing on creating modern, high-performance, and visually stunning web interfaces.

---

## Features

- **Cyber-Minimalist Design:** A premium dark-themed UI inspired by modern SaaS aesthetics.
- **Bento Grid Projects:** An advanced 12-column grid layout for projects, demonstrating mastery of CSS Grid.
- **Glassmorphism UI:** Extensive use of `backdrop-filter`, transparency, and subtle borders for a sleek look.
- **Interactive Background:** Integrated **tsParticles** for a dynamic and engaging user experience.
- **Fully Responsive:** Optimized for all devices (Mobile, Tablet, and Desktop) using a "Desktop-First" approach.
- **Modular Architecture:** CSS is split into sections for maintainability and "Clean Code" standards.
- **Smooth Navigation:** Custom smooth scroll behavior with active state management.

---

## Tech Stack

- **HTML5:** Semantic markup for SEO and accessibility.
- **CSS3:** Custom properties (variables), Flexbox, Grid, and BEM methodology.
- **JavaScript (Vanilla):** DOM manipulation, event listeners, and interactive UI components.
- **Libraries:** [tsParticles](https://particles.js.org/) for background effects and [Devicons](https://devicon.dev/) for tech stack representation.

---

## Project Structure

The project follows a modular organization to ensure scalability:

```text
├── index.html          # Main entry point
|
├── js/
│   ├── app.js          # Main UI logic & Menu interactions
│   └── particles.js    # tsParticles configuration
├── public/
│   ├── icons/          # SVGs and branding icons
│   └── img/            # Project screenshots and profile media
└── src/
    ├── css/            # Modular CSS files (header, hero, about, etc.)
    └── views/          # Secondary pages (e.g., mascotas.html)


🔗 **Live Demo:** [jorgecb12.github.io](https://jorgecb12.github.io/)
```
